Most Used

F / F11 : Fullscreen
Space : Pause or play movie
Ctrl + Arrow Up / Arrow Down : Volume

Mouse Actions

Click : Pause or play movie
Double Click : Fullscreen
Scroll : Volume (when above video) or Seek (when above progress bar)
Right Click : Context menu

Movie Navigation

M : Mute and unmute audio
P : Play movie (from the very beginning)
Esc : Exit full screen mode
[ / ] : Faster / slower
= : Normal speed
A : Aspect ratio
C : Crop screen
Z : Change zoom mode
J / K : Decrease / increase audio delay
T : Show time
E : Next Frame. Advance forward frame-by-frame

Jump in Playback

Shift + Arrow Left / Arrow Right : Very short jump (3 seconds)
Alt + Arrow Left / Arrow Right : Short jump (10 seconds)
Ctrl + Arrow Left / Arrow Right : Medium jump (1 minute)

Playlists and Special Commands

Ctrl + L : Open playlist menu
Ctrl + S : Scan server for episodes
Ctrl + H : Hide / unhide controls
N : Play next movie from playlist

Subtitles

Alt + Arrow Up / Arrow Down : Change Subtitle Size
Shift + Arrow Up / Arrow Down : Move Subtitles Up / Down on the Screen
G / H : Decrease / increase subtitle delay

Developers

Ctrl + D : Show Developer Console
